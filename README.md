Prompt program for Fish shell written in Haskell

## Install 

```
> cabal configure
> cabal build
> cabal install
```

## Features

+ show local **time**
+ show **git status** 
  + "#" for *clean*
  + "\*" for *dirty*
+ show **git branch** 
+ show **working directory**
  + be shortened if too long
